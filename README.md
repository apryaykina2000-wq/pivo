# Саша и пиво

Аркадный раннер на Phaser 3. Цель — собрать 60 предметов (пиво, бокал, рыба). За 20/40/60 очков выдаются бронзовый/серебряный/золотой кубки. Рекорд сохраняется в `localStorage`.

## Запуск локально
Откройте `index.html` напрямую в браузере **или** поднимите статический сервер.

## Деплой на GitHub Pages
1. Создайте репозиторий (например, `sasha-i-pivo`).
2. Содержимое этой папки положите в корень репозитория.
3. Включите Pages: Settings → Pages → Deploy from branch → `main` / root.
4. Откройте выданный URL.

## Управление
- Тап/клик — прыжок.
- Повторный тап в воздухе — двойной прыжок.
- Столкновение с препятствием — конец игры.
